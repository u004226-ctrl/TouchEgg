# TouchEgg

A simple clicker game where you touch an egg.

## Features

- Click on the egg to increase the counter.
- Background music.
- Simple and clean user interface.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/u004226-ctrl/touchEgg.git
   ```
2. Navigate to the project directory:
   ```bash
   cd touchEgg
   ```
3. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

1. Run the Flask application:
   ```bash
   python app.py
   ```
2. Open your web browser and go to `http://localhost:5500`.
